The input file associated with the input type chosen.

1. Project (default): The path of the project directory (*),
2. IPA: The full path including the .ipa project, and
3. APP Bundle: The full path include the .app project

* The project directory requested is relative to the workspace root. For an iOS app project, the path ends at the "*.xcodeproj" directory (e.g. "src/MyApp.xcodeproj"). For an Android app project, this is the directory containing the build.xml file.
